Introduction to Python

Course Overview

Learn how to program in Python. This is an introduction to programming course, learn concept of python and implement it in real life problems through programs.

Syllabus

Introduction to Computer Programming, Basics of Python Programming Language, Variables and Assignments, Function calls, User-defined functions, Scripts and Modules, Specification, Testing, Strings, Conditionals, Sequences, For loops,While loops, Python Memory, Errors and Asserts, Error Handling, Recursion, Nested Lists, Dictionaries.

Faculty

  • prof. Dr. Kushal Shah

Learning Outcome

  • Understand the basic concept of computer programming.
  • Understand and apply conditional statements to solve problems.
  • Understand and use various variable types and data structures in Python.
  • Understand and apply loops to solve problems.
  • Understand and apply functions to solve problems.
  • Understand and apply various useful programming practices.
  • Understand the real life problems through programming
  • Understand the mathematical concept in programs
